# 기능 정리

## 📌 서비스 정책
- 로그인하지 않으면 서비스 이용 불가
- 그룹 생성 시 초대코드 발급
- 여행별로 지출 관리

## 📋 기능 목록

| 화면 ID | 화면명 | 기능 설명 |
|---------|--------|----------|
| auth-01 | 로그인 | 로그인 하는 화면, 서비스 첫 진입 화면 |
| auth-02 | 회원가입 | 이름, 이메일, 비밀번호 입력 후 계정 생성 |
| auth-03 | 그룹생성 | 그룹 신규 생성 또는 초대코드로 기존 그룹 참여 |
| expenses-01 | 지출내역 | 날짜, 금액, 카테고리 등 지출 기록 |
| expenses-02 | 정산멤버 | 해당 지출 참여 인원 및 결제자 선택 |
| main-01 | 메인 | 메인 페이지 |
| main-02 | 여행상세 | 여행 버튼 클릭 시 상세 정보 |
| main-03 | 새 여행 목록 생성 | 제목, 기간, 최대 인원 등 추가 |
| expenseslist | 지출목록 | 카테고리별 지출목록, 사진 내역 확인 |
| stats-01 | 지출통계 | 각 여행별 지출 총계 및 결제 세부내역 |
| stats-02 | 정산 | 각 여행별 정산 결과 및 내역 |
