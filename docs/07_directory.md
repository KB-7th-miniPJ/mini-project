# 디렉토리 구조

## 📁 전체 구조
```
mini-project/
├── docs/
├── frontend/
│   ├── src/
│   │   ├── api/
│   │   ├── components/
│   │   ├── hooks/
│   │   ├── pages/
│   │   ├── router/
│   │   ├── stores/
│   │   └── utils/
│   ├── public/
│   ├── db.json
│   └── package.json
└── README.md
```

## 📌 폴더 설명
| 폴더 | 설명 |
|------|------|
| api/ | API 호출 함수 모음 |
| components/ | 재사용 가능한 컴포넌트 |
| hooks/ | 커스텀 훅 모음 |
| pages/ | 페이지 컴포넌트 |
| router/ | 라우터 설정 |
| stores/ | Pinia 상태관리 |
| utils/ | 유틸리티 함수 모음 |
