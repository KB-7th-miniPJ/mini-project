# DB 설계

## 📌 users (회원)
| 필드 | 타입 | 설명 |
|------|------|------|
| id | string | 고유 ID |
| name | string | 이름 |
| email | string | 이메일 |
| password | string | 비밀번호 |
| joinTravelIds | object | 참여한 여행 ID 목록 |

## 📌 travels (여행)
| 필드 | 타입 | 설명 |
|------|------|------|
| travelId | string | 여행 ID (자동생성) |
| inviteCode | string | 초대코드 |
| title | string | 여행 제목 |
| destination | string | 여행 목적지 |
| startDate | string | 여행 시작 날짜 |
| endDate | string | 여행 끝나는 날짜 |
| membersCount | number | 인원 수 |
| amount | number | 예산 |
| currency | string | 단위 |
| status | string | 진행상태 |

## 📌 expenses (지출)
| 필드 | 타입 | 설명 |
|------|------|------|
| id | string | 고유 ID |
| travelId | string | 여행 ID |
| date | string | 날짜 |
| category | string | 카테고리 |
| place | string | 사용처 |
| amount | number | 금액 |
| payerId | string | 결제자 ID |
| participants | array | 참여인원 ID 목록 |
| memo | string | 메모 |
| photoUrl | string | 사진첨부 |

## 📌 categories (카테고리)
| 필드 | 타입 | 설명 |
|------|------|------|
| id | string | 고유 ID |
| name | string | 카테고리명 |
| icon | string | 아이콘 |

## 📌 settlements (정산)
| 필드 | 타입 | 설명 |
|------|------|------|
| id | string | 정산고유 ID |
| travelId | string | 여행 ID |
| fromUserId | string | 송금자 ID |
| toUserId | string | 수금자 ID |
| amount | number | 액수 |
| status | string | 정산 상태 (pending → requested → completed) |
