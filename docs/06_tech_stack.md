# 기술 스택

## 🛠️ 사용 기술

| 기술 | 버전 | 선택 이유 |
|------|------|----------|
| Vue.js | 3.x | 컴포넌트 기반 구조로 화면별 역할 분리가 명확하고, 팀원 간 협업 시 코드 충돌을 최소화할 수 있어 선택 |
| Vue Router | 4.x | Vue 공식 라우팅 라이브러리로 SPA 환경에서 페이지 전환을 간결하게 관리할 수 있어 선택 |
| Pinia | 2.x | Vue 3 공식 상태관리 라이브러리로 여러 컴포넌트에서 공유되는 로그인 상태, 여행 데이터 등을 중앙에서 관리하기 위해 선택 |
| Axios | 1.x | REST API 호출 시 직관적인 문법과 에러 처리가 용이하여 선택 |
| json-server-auth | - | 별도의 백엔드 서버 없이 REST API 환경과 인증 기능을 빠르게 구현하기 위해 선택 |
